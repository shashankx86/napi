## Nest user api
backend api to control services on nest

## Setup 

- Clone repo
  ```
  git clone https://github.com/shashankx86/napi
  cd napi
  ```
- Install Deps
  ```
  go install
  ```
- Start api
  ```
  go run index.go
  ```

FOR FRONTEND SETUP REF. TO REPO https://github.com/shashankx86/ncwi-dev
